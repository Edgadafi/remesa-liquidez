| Elemento | Copy |

| Elemento | Copy |

## Diferenciadores vs competencia

| vs | Remesa TIA |
|----|------------|
| Western Union | Fee 5–8% → 0.25% on-chain |

| Strike / crypto P2P | UX técnica → TIA + comercio local |
| Apps bancarias | Lento → ~400ms settlement Solana |
